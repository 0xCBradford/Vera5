# Vera5

**Vera5** is an open-source browser extension that brings IOC enrichment to the page you are already on—built for SOC, CTI, DFIR, and threat hunting workflows.

Hover detected indicators (IPs, domains, URLs, hashes, CVEs) and review attributed context from threat intelligence sources you configure, without leaving your investigation surface.

## Local-first operation

Vera5 runs enrichment from your browser using **your own API keys**. There is no required Vera5 cloud service, no default telemetry, and no silent upload of page content—only the indicators you choose to enrich are sent to the third-party sources you enable.

You control which connectors are active, when enrichment runs, and what leaves your machine.

## Status

The extension and enrichment connectors are under active development. See `Vera5.md` for product scope and architecture notes.
